念如忆，追逐一梦
心中有一梦，梦醉不自醒，记忆深刻，爱而不得。你在世间飘零的思念，终被折磨的凄凉。过去的欣喜，变成嘶哑哼唱的小曲，日日夜夜在狂风大作的夜中消散在原野，这份不能到达时光尽头的寄托，在凋零的季节里洋洋洒洒，这是自由的心追一个人梦。

我的追逐在每一个夜里深沉，我的思念在每一个梦里沉醉，我的心在每一个初晨复苏，在等一天不能相依的吻，等一月不能问候的心，等一年不能拥抱的人。彼此的心在冰冷的夜里，相依相偎，在黑暗的相见相融。那一抹冷念在心的边缘游走，穿过那层层递进的温暖，直达心灵深处，暖的像炽热的光，穿透这夜的幕布，翻开黎明，露出阳光的脸，暖化一切冰封的往事，消融而去，让心变得晶莹剔透。

我的追逐是四野的风景，空旷的天地间那闪闪发光的中心点那么的一目了然，纵然走丢在这里，也能将你作为彼岸，一路繁花到达。我爱你就是这些喧闹的小虫子，奔放的自由花，还有我内心的独白，构成爱你的筹码，在人间痴情快意，在彼此之中连接一线。我可以摘一朵花，这万物都是你的光所育养，哪怕一点点的星辰也能将你的温柔清晰的表达，让我爱的热烈欢快。

我的追逐是你梦的笑容，它可以是任何东西，哪怕是一粒微尘，一滴雨露，一缕清风，都能让我在梦中安静。你可以游走于九天星河，也可以遨游于苍穹广宇，但是，只要你笑的真切，我便无处不在，在你胸膛里的一个细胞，在你爱中的一小撮思念，在你悲伤情绪中的一个有趣的故事，始终让你的笑百倍绽放，让你的心永远如春花灿烂。

我的追逐是你走过的小河，我可以是一棵树，可以是一条鱼，可以是你脚下的尘泥，可以是石板桥上的石子，只要你走过，我便能随你舞动，我无需多闹动，就那么静静的望着你走来的幸福和走后的快乐。你也不必刻意停留，或在我心里种下一颗种子，因为你来了，我便无处不在的开着花，等你从我身旁走过来采摘，等你纤纤玉手来触碰我的心窝，这应是我守望最期待的情节，哪怕不能也要等待的痴心。

我的追逐是晴朗明媚的清晨，我从梦里醒来，寻第一道光明，拉开窗帘，盯着远方，伸手像能触摸到那远方的过客，那人在我眼眸中生辉相映，在脑海里印成诗文，在记忆里写下悲伤与幸福的故事，但结局却没有划上句号，那么欢乐，那么幸运，一直走着一条归来乐趣的梦，知音不断，梦不醒。

我的追逐是你的全部，因为遇见相思，自始至终便都跟着一个人的脚步，求一步能正巧合适。你若是鱼，我便是水，你一直在我心中，你虽仅有七秒记忆，却要在我心中一辈子;你若是草儿，我便是雨儿，我用三千米的距离瞬间奔向你，落入你心头的每一滴，都是我爱你;你若是云朵，我便是臭氧层，我们相隔距离很远，没有我你又能逃到哪里，我隔绝宇宙保留你的温度;你若是皓月，我便是星辰，我们可以距离很远，但是只要站在地球上，我就觉得我们就能相拥在一起;你若是诗文，我便是墨笔，总能写一副挚爱，珍藏在心海，成为永恒不变的梦;你若是山河，我便是流沙，相融必定是天崩地裂，同行一定喷薄豪迈。

我爱你是孤单的开始，也是幸福的起点，是相思的凌乱，也是难耐的不安，可有一句话，将我烦躁擦去，留下一团烈火，用以燃烧数月，久不消散!我爱你是疯魔的病，也是狂癫的痛，总不能捡起你旧话的温柔来填补心中的痴念，致远不能拥有的星空，若是归来都不是你，何必温柔这个城市。
